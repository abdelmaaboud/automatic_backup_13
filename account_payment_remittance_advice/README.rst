=====================================
   Sent state on invoice
=====================================

This module adds a new state info "sent" for the invoice, it is set auto to 'True' when sent by email or printed.

This module adds a new state info "sent" for the invoice, it is set auto to 'True' when sent by email or printed.
It shows it in the tree view and in the form view.
