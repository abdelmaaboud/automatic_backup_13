Sale Service Invoicing Task Done

The purpose of this module is to create an invoice based on the Closed Task.

  • For Create an Invoice based on Task which are in Done State.

  • Set Product as Service Product and Set Invoice Policy as 'Invoice Base on Closed Task'

  • For create Task on Sale Order Create set Track Service as 'Create a task and Track Hours'.

Create Sale Order using Product which has ‘Invoice base on Closed Task’ as invoice policy

  • On Sale Order confirm, Task will be created.

  • When we change task stage, it'll reflect in Sale Order Line ( related to task ).
